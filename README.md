# Appointment Booking System

A web-based Appointment Booking System developed using:

- Java (Servlets - Jakarta)
- JSP
- JDBC
- MySQL
- Apache Tomcat 10

## Features
- Add Appointment
- View Appointments
- Update Appointment
- Delete Appointment
- Search Appointment

## Database
Database Name: appointmentdb

Table: appointments
